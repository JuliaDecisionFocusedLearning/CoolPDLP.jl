# Challenge

Folder related to the [2026 Land-Doig MIP Competition](https://www.mixedinteger.org/2026/competition/).

Inside the `data` subfolder, you can download (and decompress):

- the [instances](https://drive.google.com/file/d/1AcD7oW0grX7lWtHI0T5ke_6NeFi1kuKq/view?usp=sharing) from the test set (note that decompressing each individial `.mps.gz` file will enable faster reads)
- the [objective values](https://www.mixedinteger.org/2026/competition/instance_5m_bounds.txt) after 5 minutes of Gurobi solve
- the [solution checker](https://www.mixedinteger.org/2026/competition/miplib2017-checker.zip) from MIPLIB 2017
